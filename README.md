# tobibechtold.dev

My personal portfolio website.

Live site: [https://tobibechtold.dev](https://tobibechtold.dev)

## Tech Stack

- React
- TypeScript
- Vite
- Tailwind CSS
- shadcn/ui

## Local Development

```bash
npm install
npm run dev
```

## Build

```bash
npm run build
```

## Preview Production Build

```bash
npm run preview
```

## Deployment

This project is deployed on Vercel.
